Introduction



In this learning unit, we will do a coding challenge similar to those you will have in a technical interview.

In order to train you for those important moments, we have created a Codewars kata.

How to confront the interview

It is important to think about the methodology as well, not only to deliver the first working code you can think of.

Remember in a coding interview a lot of people can reach a possible solution, so you have to make your best effort to stand out.

Kata description

We are going to do the kata Directions Reduction.



You have limited time to complete the code and pass all the test included in the kata.

Read the description of the problem carefully and pay attention to the required returning value.

Finally, deliver your solution in Codewars and compare it with the community and classmates solutions to learn about different approaches.
